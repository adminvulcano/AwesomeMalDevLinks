Armory

Rust Beacon Object Files (BOFs) for adversary simulation, threat emulation, security research, and detection engineering. All 115 TrustedSec BOFs ported from C to Rust using the rustbof framework.

Credits

  • MITRE ATT&CK - Adversarial tactics, techniques, and common knowledge framework
  • rustbof by Joao Victor - Rust BOF framework
  • CS-Situational-Awareness-BOF by TrustedSec - Original C BOFs (Situational Awareness)
  • CS-Remote-OPs-BOF by TrustedSec - Original C BOFs (Remote Operations and Injection)

Building

Requires Rust nightly, boflink, cargo-make, and MinGW-w64.

cd bofs/sa/whoami
cargo make
# Output: out/whoami.x64.o

Use COFFLoader or any compatible loader to test.
