# Ligolo-IWA

[Permalink: Ligolo-IWA](https://github.com/nicocha30/ligolo-iwa#ligolo-iwa)

A Ligolo-ng JavaScript agent working inside Chrome/Chromium-based browsers by leveraging Isolated Web Applications.

[![Ligolo-IWA](https://github.com/nicocha30/ligolo-iwa/raw/master/docs/demo.png)](https://github.com/nicocha30/ligolo-iwa/blob/master/docs/demo.png)

See the Ligolo-IWA doc for quick setup: [Ligolo-IWA (Setup/Quickstart)](https://iwa.ligolo.ng/)

And don't forget to read the Ligolo-ng documentation: [📑 Ligolo-ng Documentation (Setup/Quickstart)](https://docs.ligolo.ng/)

## Local setup

[Permalink: Local setup](https://github.com/nicocha30/ligolo-iwa#local-setup)

Build the signed Isolated Web App bundle locally with:

```
make
```

The signed artifact is written to `out/ligolo-iwa.swbn`.

## Install on Chrome

[Permalink: Install on Chrome](https://github.com/nicocha30/ligolo-iwa#install-on-chrome)

Chrome's current IWA testing flow uses Web App Internals:

1. Use Chrome 120 or later.
2. Enable `chrome://flags/#enable-isolated-web-app-dev-mode`.
3. Open `chrome://web-app-internals`.
4. Choose `Install IWA from Signed Web Bundle`.
5. Select `out/ligolo-iwa.swbn`.

## Install on Edge

[Permalink: Install on Edge](https://github.com/nicocha30/ligolo-iwa#install-on-edge)

Due to current limitations in how Microsoft Edge handles the new Isolated Web App standard, installation is currently unsupported on this browser.
